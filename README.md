**Status do projeto:** Finalizado✅ (Possíveis melhorias no futuro)

## Tecnologias utilizadas
- Java (JDK 8 ou superior)
- Java Swing (Interface gráfica)
- MySQL (Banco de dados)
- JDBC (Conexão com o banco)
- NetBeans IDE

## Time de desenvolvedores
- Igor.

## Objetivo do software
Desenvolvido um sistema desktop para agendamento de serviços (salão, barbearia, clínica, etc.), permitindo gerenciar clientes, serviços, profissionais e agendamentos de forma eficiente.

## Funcionalidades principais
- CRUD de Clientes, Serviços e Agendamentos
- Combos dinâmicos (atualizam automaticamente após novos cadastros)
- Tela de Agendamentos Cadastrados com filtro em cascata (Serviço → Cliente → Data/Hora)
- Validações de campos obrigatórios e formatos
- Interface com tema escuro

## Tecnologias
- Java Swing
- MySQL + JDBC
- Git + GitHub para versionamento

## Como executar
1. Importar o projeto no NetBeans
2. Configurar a conexão com o banco no arquivo `Conexao.java`
3. Executar a classe `TelaPrincipal.java`
